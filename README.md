# Sattva Sanctum 🌿

> **Where Travel Becomes Transformation**

[**Visit the Live Website! 🚀**](https://sattva-sanctum.vercel.app/)

Sattva Sanctum is a modern web platform designed to showcase curated life experiences across India—including spiritual travel, forest living, adventure, and culture. We design journeys that change how you breathe, relate, and return home.

## Tech Stack 💻
- **Framework:** [Next.js](https://nextjs.org/) (React)
- **Styling:** [Tailwind CSS](https://tailwindcss.com/)
- **Hosting & Deployment:** [Vercel](https://vercel.com/)
- **Analytics & Performance:** Vercel Web Analytics & Speed Insights
- **Fonts:** UI Fonts from Next/Font (Cormorant Garamond & DM Sans)

## Local Development 🛠️

To spin up this project on your local machine, follow these steps:

1. **Clone the repository:**
   ```bash
   git clone https://github.com/nyalamrithwik-oss/sattva-sanctum.git
   ```

2. **Navigate to the project folder:**
   ```bash
   cd sattva-sanctum
   ```

3. **Install the dependencies:**
   ```bash
   npm install
   ```

4. **Start the development server:**
   ```bash
   npm run dev
   ```

5. Open [http://localhost:3000](http://localhost:3000) with your browser to see the result!
